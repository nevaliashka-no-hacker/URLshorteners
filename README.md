# Цель:
Разработать программу "Сокращатель ссылок".

# Требования к программе:
1 При вводе URL возвращать короткий URL
2 По короткому URL возвращать оригинальный
3 Считать количество переходов
4 Максимальное количество URL, хранимых в программе, будет задаваться на входе (при начале работы). 
  Обеспечить удаление URL при достижении лимита в следующем порядке:
  а) меньшее количество использований
  б) больший возраст создания.
